Regras do Jogo

  • O jogador deve acertar o barril com a resposta correta em relação a equacao que aprece no topo da tela.
  • Para movimentar o canhão o jogador devará usar "a" para ir para esquerda e "d" para a direita.
  • Aperte a tecla "e" para atirar com o canhão !
  • O jogador terá 20s para resolver as fases fáceis
  • O jogador terá 40s para resolver as fases difíceis
  • Aperte a tecla "e" para atirar com o canhão !
  • Para as fases fáceis, necessita conhecimento básico de matemática
  • Para as fases difíceis, necessita conhecimento de cálculo 1
